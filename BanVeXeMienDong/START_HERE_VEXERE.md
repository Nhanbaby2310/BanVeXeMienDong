## **🎨 Customization Examples**

### **Change Button Color**
Edit `wwwroot/css/search-results-vexere.css`:
```css
.choose-btn {
    background: #yourcolor; /* Change color */
}
```

### **Change Sidebar Width**
```css
.search-results-container {
    grid-template-columns: 350px 1fr; /* Change 350px */
}
```

### **Add New Filter**
Edit `Views/Ticket/SelectRoute.cshtml` and add filter section in sidebar

### **Change Amenity Tags**
Edit the amenities section in result cards
