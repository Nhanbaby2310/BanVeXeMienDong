## 🎬 Animation Timings

```
Flash Sale Pulse:
  Duration: 2 seconds
  Effect: scale(1) → scale(1.05) → scale(1)
  Timing: infinite loop

Card Hover:
  Duration: 0.3 seconds
  Effect: translateY(-8px)
  Timing: ease

Swap Button:
  Duration: Variable (on hover)
  Effect: rotate(0deg) → rotate(180deg)
  Timing: ease
```

## 🎭 Component States

### Search Input

### Buttons
```
Default:  Solid color, shadow 0 4px 16px

Disabled: Opacity 0.5, cursor not-allowed
```

### Cards
```

Hover:    Border #2d5a8c, Transform Y -8px, Shadow enhanced
```

## 🎨 Visual Hierarchy

### Important (Hero Section)
- Large text (32px+)
- Bright colors (Red, Yellow)
- High contrast
- Central positioning

### Secondary (Features)
- Medium text (18px)
- Neutral backgrounds
- Cards/sections
- Grid layout

### Tertiary (Details)
- Small text (12-13px)
- Gray colors
- Labels/descriptions
- Right-aligned or muted

## 🎯 Quick Visual Tips

1. **Keep white space** - Don't crowd elements
2. **Use colors purposefully** - Red for action, Blue for trust
3. **Hierarchy matters** - Large titles, smaller body text
4. **Shadows add depth** - Use subtle shadows, not harsh
5. **Animation adds life** - Keep smooth and purposeful
6. **Mobile first** - Design for small screens first
7. **Consistent spacing** - Use a spacing scale
8. **Test contrast** - Ensure readable text
9. **Use gradients wisely** - They add modern feel
10. **Smooth transitions** - Always use 0.3s ease
